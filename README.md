# celine's semester project

The project is a website for a science museum for kids and families
this was made with CSS and HTML and it shows exhibits, visiting hours, info
and other information about the museum.

## live demo
https://celds.github.io/Semester-project-celineD/

## Features

- A simple website showing off events, exhibits and info
- hamburger menu

## Prerequisites

- Node.js (v20+)
- npm

## Getting Started

- Install Live Server extension
- Click “Go live” and it will open automatically

### Installation

```bash
npm install
```

### Running the project

```bash
npm run dev
```

### Running tests

```bash
npm run test
```

## Environment Variables

Create a `.env` file in the root directory:

```bash
API_KEY=your-api-key-here
BASE_URL=https://example.com/api
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run test` - Run tests
- `npm run lint` - Run ESLint
